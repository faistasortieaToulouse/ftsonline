# **App Name**: FTS Online Events

## Core Features:

- Event Display: Display upcoming events in a list or calendar format.
- Geographic Limiter Tool: Uses a tool that checks event locations to ensure they are within Haute-Garonne's 586 communes.
- Automatic Deletion of Expired Events: Automatically remove events from the display once their dates have passed.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel, inspired by the reliability and importance of keeping track of events.
- Background color: Light gray (#ECEFF1), offering a clean and neutral backdrop that ensures readability and focus on the events.
- Accent color: Vibrant orange (#FF9800) to highlight key event details and call-to-action buttons, analogous to the primary but brighter.
- Headline font: 'Space Grotesk' sans-serif for headlines, providing a techy and scientific feel. Body font: 'Inter' sans-serif for body, for a clean and readable style.
- Use simple, clear icons to represent event categories or types.
- A clean and organized layout with clear sections for event details and dates.
- Subtle transitions when switching between event views (list, calendar).
